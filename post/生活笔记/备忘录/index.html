互联网时代·个人隐私安全

  1. 输入法禁止联网，或者使用其他单机输入法

    • macos: 系统偏好设置->安全性与隐私->开启防火墙->防火墙选项->阻止传入连接【不好用，无法设置默认允许所有传入连接，只设定某个应用阻止连接】

    • 小米手机：设置->应用设置->应用管理->输入法app【不用系统自带的输入法，因为无法禁用WLAN联网】

  2. 浏览器使用chrome

  3. 不建议安装软件名单：国家反诈中心APP、MIUI13

老人与小孩的手机安全

  1. 优先购买 iphone 手机，相比 android 安全
  2. 设置短信拦截和来电拦截：小米系统可以直接拦截有 推销/诈骗 标记的号码
  3. 应用权限管理：禁用 读取剪贴板/读取应用列表

iPad mini 蓝牙键盘操作技巧

每次使用键盘时需要手动开机，使用完毕则需要手动关机

  1. 开关机按钮：长按开机，长按关机
  2. fn + caps: 可以切换中英文
  3. fn + =: 可以按出等于号
  4. fn + shift + 加号: 可以按出加号键
  5. cmd + c: 复制；cmd + v: 粘贴；cmd + x: 剪贴

MACOS 系统录屏

下载并安装 Soundflower

只录制系统声音

  1. 点击屏幕右上角喇叭图标，输出设备选择 Soundflower（2ch），
  2. 打开「截屏」，更改「选项」中「麦克风」一栏的设置：选择 Soundflower（2ch），最后点击录屏即可，
  3. 退出截屏快捷键：control+command+esc。

同时录制系统声音和麦克风声音

  1. 点击屏幕右上角喇叭图标，输出设备选择聚焦设备，
  2. 打开「截屏」，更改「选项」中「麦克风」一栏的设置：选择聚焦设备，
  3. 退出截屏快捷键：control+command+esc。
