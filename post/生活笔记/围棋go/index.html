围棋的英文名有三种：中国读 Weiqi，西方读 Go，韩国读 baduk(发音：怕度)。

围棋对局一般分为三个阶段：布局、中盘、收官。收官阶段也称为官子阶段。然而，这三个阶段之间并没有明显的界限，何时进入中盘，何时进入收官，完全取决于棋局的发展和对局者的决断。 围棋中盘战术的主要技术，除了包括计算能力、掌控全局能力和形势判断之外，还包括进攻、防守、手筋、死活、打入、侵消等多方面的战术。

李昌镐说过一句名言，“围棋不是看你学了多少，而是看你会用多少。”

初学者

初学者注意：千万不要下简单人机，否则会被胖揍到失去自信，一定要和真人下！！！

初学者可以先去下9路棋盘，9路棋盘主要是练习对杀、死活和官子等基本功。

9路棋盘虽小但变化复杂，开局优先占星位(如4,4)、高目(如4,5)，其次是目外(如5,5)。【黑先手一般先占领天元】

围棋礼仪：执黑一方将第一手棋下在自己的右上角，因为这是伸手最方便的一个地方，同样也将对手最方便的地方留出，“这不仅是礼貌，还是换位思考”。

终局计算：数子法和比目法

当今世界，主要玩围棋也就中日韩三国，其中中国采用数子法，日韩采用比目法。

日韩比目法只要黑棋贴6目半，而中国数子法要黑棋贴3又3/4子。

总结一下，不管是按中国数子规则，还是日韩点目规则，对围棋胜负的判断结果是一样的，大家可以放心地交换使用。但是要说这俩完全一样的话需要有一个很苛刻的条件，就是黑白双方的手数一样，也就是说不能是黑棋收后，也不能有任何一方停招。

规则简单说明

数子法：

如果对局双方是平下，则终局后先将双方死子全部清理出盘外，然后对一方的活棋（包括活棋围住的点）以子为单位进行计数。 双方活棋之间的空点各得一半，一个点即为一子。 就现行的中国围棋规则来看，黑方超过184又1/4为胜，反之为负(注：平下不可能是和棋）。 (184.25 = 180.5 + 3.75)

如果是让子棋，则黑方在终局时还给白方让子数的一半，随后数棋。若本局是黑收后，则黑棋要超过181子算胜。若本局是白收后，则黑棋要超过180又1/2子算胜（注：可以是和棋）。

  1. 单官收完。双方确认无误，同意棋局终了。
  2. 然后剔除盘面上，不活的孤子。
  3. 再来在双方的领地上，填满己方的棋子。
  4. 数双方的棋子数目，再加上规定的贴子数目，就可以判定输赢。

比目法：

不用把死棋去除，而是将棋放入本方空内（如：白方空内的死黑棋放入黑方空内）。将每方空内的空格数（即无子处）相加，判定胜负（目前规则是黑贴6目半）。

  1. 官子收完，领地划分完，双方确认无误，同意棋局终了。（可不收完单官）
  2. 然后将对方不活的孤子，填入对方的领地。
  3. 再把对方死子也同样的填入对手的领地。
  4. 最后数双方所围的空，再加上规定的贴目数目，就可以判定输赢，数子法规则比较简单，也比较没有争议，而比目法就比较复杂，有很多特别棋型需要沿用判例加以辅助。例如：像是盘角曲四。

规则详细说明

终局算法 黑 白
数子法 黑活子+黑空 白活子+白空
比目法 黑空-黑死子 白空-白死子

对两列表格分别做减法就会发现，比目法其实是在数子法的基础上减去了（活子+死子）。而活子+死子其实就是一共下在棋盘上的手数。也就是说，数子法得出的结果和比目法一样当且仅当黑白双方总手数一样。由于围棋是一人一手下的，因此双方的总手数应该是差不多的，所以我说这两种数法大体上是一样的，所以您一般不用担心因为换了一种数法结果就不同了。由于比目法得出的数字更小，所以对局时的形势判断一般都是用比目法。

传统打谱方式介绍

什么叫打谱？

打谱，就是按照高手下出的棋局的棋谱，把棋子在棋盘上按顺序再摆一遍。打谱过程中，可以感受高手下出棋的奇妙，可以停下来，自己研究，可以培养自己的大局观，等等，打谱也是长棋的一种方式。

有效的打谱方法

一、快速打谱

就是你照着棋谱能摆多快就摆多快，这种方法可以提高你的棋感。一段时间之后你会觉得下棋时很多地方一眼就找到了要走在哪里才好。但要注意这种方法一定要选择顶尖高手的棋谱，并且这个棋手的风格应该是稳健的，

二、超慢速打谱

这种打谱方法就是要详细的研究每个变化，每一步棋。要找到那种吃完晚饭开始打谱，等打完谱抬起头来发现天已经亮了的感觉。这种方法选择的棋谱就可以是任何棋风的了。只要是喜欢想研究的棋手的都行。

打谱的目的

  1. 学习对局者一些着法，主要是技战术方面；
  2. 学习一些布局知识、定式和套路，特别是当下较流行的具体走法；
  3. 入门者通过打谱，模仿高手的序盘行棋步调，从纠缠局部开始转到把棋走开。
  4. 通过一手执书，一手落子，坐直身姿，气定神闲来提升个人的气质和休养。

打谱的方法

  1. 根据打谱人的水平，由少至多的来打。初学者可以摆50手左右；中级水平可以打100手左右，高级者可以通盘来打。
  2. 谱中的解说一般不用全部细看，最好是遇到不甚理解的着法时，可以看解说来解惑。

围棋术语

手筋

手筋，围棋术语，出自日本，意思是局部对弈中的最佳妙招。日本棋道中的“筋”与中国的原意最为接近，为“精髓的”、“要害的”之意。棋中有“宁舍片子，不舍一筋”的说法。

在围棋中，“筋”意指棋子与棋子之间的关系。在所有的筋之中，效果极大的特殊手法，即称为手筋。

常见手筋：双打、枷、扑、倒扑、接不归[扭羊头]、挂、尖、顶（鼻顶、尖顶）、长（立、冲、退、爬、贴、并）、扳、粘、跳、飞、挡、靠（碰、压、托）、虎、挖、挤、拐、跨、镇（封）、穿象眼、刺（点、觑）、夹、接、拆、逼、渡

官子

「单官」，就是占不到“目”的一手棋，只是用棋子占据棋盘上的一个交叉点。单官一般都是最后才去抢占的。

根据黑方或白方谁先落子，以及落子后谁是先手谁是后手，官子可以分为4类：「双先官子」「单先官子」「逆先官子」「双后官子」。

正确的收官顺序：「双先单先无限抢，逆先双后看价值」是最佳策略

其他术语

妙手、本手、俗手、恶手

围空、外势，实地、滚打包吃、抢先，脱先，不入气

进攻战术：缠绕进攻、声东击西、借力发力

模样：所谓模样，指尚未形成确定地域的大阵势。

净活：若干子虽被包围，但能做成两个（或两个以上）眼，无条件地成活，称为“净活”。净杀：即净吃，无法成活。

效率：围棋最终比较的是双方领地的大小，所以在对局中，每一手棋都是以多占领一些领地为目标。一手棋在围空上发挥的作用越大，我们就说这手棋的效率越高。如果能在某个重要的局部连下两手棋，那么这两颗棋子的效率通常会成倍增加。

急所：来源于日本围棋术语，现在我国棋界也广为使用了。它是指在敌我双方互相接触的棋形中，无论对于攻方还是守方都互关重要的那些点。

缓招：在围棋中，步调慢，效率不高的棋叫做缓招，缓招一般可以暂时不走，也就是不是“急所”。实战中走出缓招，会被对手抓住机会，痛下杀手的。

计算力和大局观，是下好围棋的两大技能。计算力，是指遇到局部战斗时通过对下法的推算找到局部最佳下法的能力。吃子、死活和对杀的知识，都是计算力的核心部分。大局观，是指在全局范围内通过对各个部分的价值判断找到全局最佳下法的能力。布局、定式、中盘和官子的知识，都是大局观的核心组成部分。

孤棋，围棋术语，对局中，一方在对方势力的威胁下尚未生根安定的棋子。治孤，就是巧妙利用对方的棋型缺陷和薄弱环节，将自己的孤棋进行妥善、高效的处理，“治孤法”是一种围棋高级战术。

勺子（漏勺）：指的是围棋对局中出现明显漏招，误算，招至不必要的损失。

进攻与防守

「封锁」和「搜根」是攻击的主要手段。

想要杀死一块孤棋，需要完成两件事：「封锁」和「搜根」。

「封锁」是指堵住孤棋向中腹逃跑的方向，将孤棋困在角部或边上；

「搜根」是指缩小或破坏孤棋在角部或边上的做眼空间，使孤棋无法做成两只真眼。

「出头」和「生根」是防守的主要手段。

想要加强一块孤棋，可以做两件事情：「出头」和「生根」。

针对封锁，「出头」是指将孤棋向中腹延伸，保证逃跑路线的畅通；

针对搜根，「生根」是指扩大在角部或边上的做眼空间，使孤棋做成或随时可以做成两只真眼。

「巩固」和「扩张」,「打入」和「侵消」

当中盘阶段找不到可以进攻的孤棋时，中盘的主要内容就是直接争取利益，也就是地盘的争夺。

对于自己的地盘，可以采用「巩固」和「扩张」两种方法：「巩固」是指堵住地盘的缺口，防止对方入侵；「扩张」是指将地盘的范围扩大，争取围住更大范围的地盘。

对于对方的地盘，可以采取「打入」和「侵消」两种方法：「打入」是指深入对方的地盘，成为对方地盘中的孤棋，但可以彻底破坏对方的地盘；「侵消」是指压缩对方地盘的范围，阻止对方扩张，使对方只能采取巩固的方法围空。

手筋题和死活题

做死活题的规律，一般是一冲、二扳、三点杀。

手筋题和死活题的主要区别有三个：

一是开放性不同，死活题往往限定在一个封闭的局部，可选的点一般不多；而手筋题往往给出的是开放的区域，选点通常要比同等难度的死活题多，因此在缺乏经验积累的情况下，需要考虑的点位就多。

二是思维步骤不同，死活题预先给出了目标，只需要考虑做活或者杀棋的手段就行了，不需要有任何战略性的考虑；而手筋题先要评估得失，然后根据对局面的评估选择目标，比如选择逃出，还是弃子之类，然后才是考虑具体的手段。即使有些题目为了降低难度，在题目分类上指明了手筋的类型，比如说封锁的手筋，但具体应该采用什么手段去封锁，很多时候依然要根据局面去评估。潜在选点可能更多，思考步骤也更多，有时候觉得无从下手也就很正常了。

三是目标不同。这一点其实是最重要的。做死活题的时候，虽然顺便也会记住一些常见死活，但最主要的目标是培养计算力；而手筋题的最终目标其实是培养第一感——虽然一开始会觉得选点众多，但随着经验的积累，看到类似的局面时，应该可以立即把选点缩减至很有限的几个，然后根据对全局的判断，仅在这几个选点中进行计算。而这虽然并不提升你的计算力，但会大大提升你的计算效率，从而进一步提升你的计算深度——在后续计算的每一步中，都减少在无效选点上的计算消耗。

围棋棋理

  • 三思而行：道生一，一生二，二生三，三生万物。此三为多之意。围棋的本质是三步计算法，每手棋至少计算3步（即每手棋必须考虑3个及以上的点进行计算），然后对比，选择获利最多的变下。
  • 布局结束后，不要为了围空而围空，要优先进攻孤棋，在双方交手中想方设法围出更多空。
