围棋对局一般分为三个阶段：布局、中盘、收官。收官阶段也称为官子阶段。然而，这三个阶段之间并没有明显的界限，何时进入中盘，何时进入收官，完全取决于棋局的发展和对局者的决断。 围棋中盘战术的主要技术，除了包括计算能力、掌控全局能力和形势判断之外，还包括进攻、防守、手筋、死活、打入、侵消等多方面的战术。

李昌镐说过一句名言，“围棋不是看你学了多少，而是看你会用多少。”

初学者

初学者注意：千万不要下简单人机，否则会被胖揍到失去自信，一定要和真人下！！！

初学者可以先去下9路围棋，9路棋主要是练习对杀、死活和官子等基本功。

9路棋虽小但变化复杂，开局优先占高目(如4,5)，其次是目外(如5,5)，最后才是星位(如4,4)。

围棋礼仪：执黑一方将第一手棋下在自己的右上角，因为这是伸手最方便的一个地方，同样也将对手最方便的地方留出，“这不仅是礼貌，还是换位思考”。

终局计算：数子法和比目法

当今世界，主要玩围棋也就中日韩三国，其中中国采用数子法，日韩采用比目法。

日韩比目法只要黑棋贴6目半，而中国数子法要黑棋贴3又3/4子。

总结一下，不管是按中国数子规则，还是日韩点目规则，对围棋胜负的判断结果是一样的，大家可以放心地交换使用。但是要说这俩完全一样的话需要有一个很苛刻的条件，就是黑白双方的手数一样，也就是说不能是黑棋收后，也不能有任何一方停招。

规则简单说明

数子法：

如果对局双方是平下，则终局后先将双方死子全部清理出盘外，然后对一方的活棋（包括活棋围住的点）以子为单位进行计数。 双方活棋之间的空点各得一半，一个点即为一子。 就现行的中国围棋规则来看，黑方超过184又1/4为胜，反之为负(注：平下不可能是和棋）。 (184.25 = 180.5 + 3.75)

如果是让子棋，则黑方在终局时还给白方让子数的一半，随后数棋。若本局是黑收后，则黑棋要超过181子算胜。若本局是白收后，则黑棋要超过180又1/2子算胜（注：可以是和棋）。

  1. 单官收完。双方确认无误，同意棋局终了。
  2. 然后剔除盘面上，不活的孤子。
  3. 再来在双方的领地上，填满己方的棋子。
  4. 数双方的棋子数目，再加上规定的贴子数目，就可以判定输赢。

比目法：

不用把死棋去除，而是将棋放入本方空内（如：白方空内的死黑棋放入黑方空内）。将每方空内的空格数（即无子处）相加，判定胜负（目前规则是黑贴6目半）。

  1. 官子收完，领地划分完，双方确认无误，同意棋局终了。（可不收完单官）
  2. 然后将对方不活的孤子，填入对方的领地。
  3. 再把对方死子也同样的填入对手的领地。
  4. 最后数双方所围的空，再加上规定的贴目数目，就可以判定输赢，数子法规则比较简单，也比较没有争议，而比目法就比较复杂，有很多特别棋型需要沿用判例加以辅助。例如：像是盘角曲四。

规则详细说明

终局算法 黑 白
数子法 黑活子+黑空 白活子+白空
比目法 黑空-黑死子 白空-白死子

对两列表格分别做减法就会发现，比目法其实是在数子法的基础上减去了（活子+死子）。而活子+死子其实就是一共下在棋盘上的手数。也就是说，数子法得出的结果和比目法一样当且仅当黑白双方总手数一样。由于围棋是一人一手下的，因此双方的总手数应该是差不多的，所以我说这两种数法大体上是一样的，所以您一般不用担心因为换了一种数法结果就不同了。由于比目法得出的数字更小，所以对局时的形势判断一般都是用比目法。

那么究竟什么情况下双方手数会不一样呢？下面列举几种比较常见的情况。

  1. 黑先收后。

    黑先收后是最常见的导致双方手数不一样的情况，平均每两盘就会发生一盘。由于黑棋先走，如果最后一步单官也是黑棋走的话，黑棋的手数就会比白棋多一手。所以假如不贴目/子的话，黑棋又占了一点便宜。据说唐宋时期是不承认黑先收后的利益的，后来到明清时期为了避免频繁出现和棋又予以承认。

    下面我们说说为什么日韩规则只要黑棋贴6目半，而中国规则要黑棋贴3又3/4子。棋盘一共361个点，是一个单数，所以如果盘面目数差是双数就是黑棋收后，反之就是白棋收后。如果黑棋盘面7目，在日韩规则中是黑棋赢了，但是在中国规则中黑棋输了，因为盘面7目是白棋收后。中国规则需要黑棋盘面8目才赢，这就是为什么我们有时候说贴3又3/4子相当于贴7目半，当然黑棋的压力就比贴6目半更大了。可是，如果中国规则改成贴3又1/4子，那么黑棋只要盘面6目就赢了，也就是说只相当于贴5目半了，黑棋又太轻松了。因此尽管普遍反映贴3又3/4子对黑棋压力较大，但在数子法下也可以说是一种无奈之举，因为数子法不能实现让黑棋贴6目半。

    盘面 收后 日韩 中国
    黑多5目 白 白胜1目半 白胜1又1/4子
    黑多6目 黑 白胜半目 白胜1/4子
    黑多7目 白 黑胜半目 白胜1/4子
    黑多8目 黑 黑胜1目半 黑胜3/4子

    另外需要特别注意的是，如果棋盘上出现一个有眼双活这样的特殊情况，有一个点双方都无法占，那么棋盘就变成双数了。这时候贴3又3/4子就和贴6目半一样。也就是说，出现（单数个）有眼双活的情况，黑棋就便宜了1目棋，或者说白棋亏了1目棋。

  2. 让子棋

    让子棋中显然也会导致双方的手数不同，因为黑棋先在棋盘上连摆了几个子然后才一人一手开始下。但是由于一般让子棋双方实力差距比较大，而且让子棋的输赢结果一般并不重要，所以这点影响往往被忽略了。实际上，让子棋中的白棋如果采用数子法的话比比目法更难获胜，这一点也很容易明白。有一种解决方法：如果是让n子棋，那么黑棋应该贴还n/2子给白棋，这样就基本与比目法结果一致。但是我们也完全可以认为数子法才真正反映了谁嬴谁输，这并没有对错之分。但我们需要知道的是，这里数子法和比目法结果不一样。

  3. 粘劫收后

    在数子规则中，有一种特殊的手段叫做粘劫收后。也就是，当棋盘上还剩在我手里的劫和若干单官的时候，我不先粘劫，而先收单官，在此过程中仗着劫材优势一直撑着这个劫。当我收完最后一个单官，再来提劫，迫使对方由于没有劫材而停招，然后我再把劫粘上。之所以叫做特殊手段，是因为它需要我方有大量的劫材（多于单官数量的一半加上对方的劫材数），而且单官的数量必须是单数。尽管如此，比赛中依靠粘劫收后逆转获胜的例子也是屡有发生。

    粘劫收后从“目”的角度来说没有任何便宜，因此日韩规则下没有这种手段。但是，当对方由于没有劫材而停招的时候，双方手数的平衡就被打破了，也就是说通过这种手段可以获取“子”的便宜，特殊情况下甚至可以便宜好几个子。

    数子法在这里赋予围棋更多变化，而比目法则相对比较稳定。

  4. 盘角曲四

    “盘角曲四，劫尽棋亡”，这句话会下围棋的人应该都知道（如果您不知道的话，现在知道了）。盘角曲四这个棋型的特殊性在于，盘角曲四尽管本身是打劫，但杀棋的一方可以选择开劫的时机，而且还是先手劫。这就意味着他可以等到把对方的劫材全部补掉，而后再行开劫，让对方没有还手的余地。但是，你在自己空里补劫材的时候，对方可以停招不用补，这就又使得双方的手数产生了差异。在自己空里走子，对于数子法来说没有任何影响，但对于比目法来说走一个就损1目。为了使得结果跟数子法一样，日韩规则均规定，一旦出现盘角曲四的棋例，就视作净死，也就是说不用去补劫材再把它吃掉。本来事情到这里也就算圆满解决了，可是没想到竟然有些劫材是补也补不掉的，比如双活。你总不能为了补劫材先把自己的双活送死吧？既然你不肯，那么对方就有可能把自己的双活送死当劫材。也就是说，按照中国规则对盘角曲四实战解决的判定，如果棋盘上有双活的话，盘角曲四的一方是有可能取得一定的代偿，甚至做活的。比目法把盘角曲四判定为死棋的解决方案在这些问题上就太死板了。幸好职业比赛中几乎没有出现过既有盘角曲四又有双活的情况。

传统打谱方式介绍

什么叫打谱？

打谱，就是按照高手下出的棋局的棋谱，把棋子在棋盘上按顺序再摆一遍。打谱过程中，可以感受高手下出棋的奇妙，可以停下来，自己研究，可以培养自己的大局观，等等，打谱也是长棋的一种方式。

有效的打谱方法

一、快速打谱

就是你照着棋谱能摆多快就摆多快，这种方法可以提高你的棋感。一段时间之后你会觉得下棋时很多地方一眼就找到了要走在哪里才好。但要注意这种方法一定要选择顶尖高手的棋谱，并且这个棋手的风格应该是稳健的，

二、超慢速打谱

这种打谱方法就是要详细的研究每个变化，每一步棋。要找到那种吃完晚饭开始打谱，等打完谱抬起头来发现天已经亮了的感觉。这种方法选择的棋谱就可以是任何棋风的了。只要是喜欢想研究的棋手的都行。

打谱的目的

  1. 学习对局者一些着法，主要是技战术方面；
  2. 学习一些布局知识、定式和套路，特别是当下较流行的具体走法；
  3. 入门者通过打谱，模仿高手的序盘行棋步调，从纠缠局部开始转到把棋走开。
  4. 通过一手执书，一手落子，坐直身姿，气定神闲来提升个人的气质和休养。

打谱的方法

  1. 根据打谱人的水平，由少至多的来打。初学者可以摆50手左右；中级水平可以打100手左右，高级者可以通盘来打。
  2. 谱中的解说一般不用全部细看，最好是遇到不甚理解的着法时，可以看解说来解惑。

围棋术语

手筋

手筋，围棋术语，出自日本，意思是局部对弈中的最佳妙招。日本棋道中的“筋”与中国的原意最为接近，为“精髓的”、“要害的”之意。棋中有“宁舍片子，不舍一筋”的说法。

在围棋中，“筋”意指棋子与棋子之间的关系。在所有的筋之中，效果极大的特殊手法，即称为手筋。

常见手筋名称：双打、扑、倒扑、接不归[扭羊头]、尖、长（立、冲、退、爬）、扳、粘、跳、飞、挡、靠（压、托）、虎、挖、挤、拐

其他术语

围空、外势，实地、官子、单官

进攻战术：缠绕进攻、声东击西、借力发力

模样：所谓模样，指尚未形成确定地域的大阵势。
