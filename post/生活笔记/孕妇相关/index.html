孕妇忌讳

  1. 不能吃：龙眼，薏米，山楂，人参，凉茶，祛湿茶等
  2. 有甲减的孕妇不能吃海苔、紫菜、瑶柱、海虾等海产品

甲减孕妇注意

优甲乐过量可能导致的问题：心慌、手抖、易饥饿、多汗、暴躁，如出现这些症状，应减少药量

重要：分娩前的最后一次检查，一定要咨询甲状腺医生，生完宝宝后，因为月子期间不能来检查，优甲乐在月子期间如何服用，药量多少。

在分娩住院期间，一定要先告诉医生的自己有甲减。

临产征兆

首先，我们要知道临产的征兆主要有见红、宫缩、破水，不管先出现哪种，孕妈都不会马上就能把宝宝生出来的，通常还要6-12个小时，甚至更久。

  1. 内裤见红：如果出现的是新鲜的血液，超过生理期的出血量，还伴随着腹痛，那么就要立马入院了
  2. 阵痛宫缩：如果孕妈感受到有规律的宫缩，然后是一阵紧接着一阵，时间间隔逐渐变短，那么做好准备，胎宝宝要出来了。
  3. 羊水破了：见红和阵痛都需要进行观察再决定是否入院，而破水不一样，只要有羊水流出，不管颜色性状一定要 立马入院 。破水也不是每个孕妈都会经历的。

产后注意

  1. 产后一个月女性会比较敏感，容易得抑郁症，要特别注意谅解她，哄她开心
  2. 产后会有妊娠纹
  3. 孩子需要补充维生素，加强钙的吸收
  4. 产后恶露是指产后随子宫蜕膜脱落，含有血液、坏死蜕膜等组织经阴道排出，称为恶露，这是产妇在产褥期的临床表现，属于生理性变化。恶露有血腥味，但无臭味，其颜色及内容物随时间而变化，一般持续4～6周，总量为500ml。如超出上述时间仍有较多恶露排出，称之为产后恶露不尽。

住院生娃需带

  1. 产妇用的：计量巾5条，产后卫生巾，护理垫，牙刷牙膏，吸管，出院一套衣服，抽血管2支
  2. 婴儿用的：纸巾干的湿的，纸尿裤，洗脸盆，出院一套衣服，抱被2张
  3. 资料：生育保险单、身份证、医保卡
  4. 住院陪护：可租折叠床，15一个晚上

分娩当天

  1. 孕妇生当天，要先洗澡洗头后再去医院。
  2. 体力下降，很容易发冷，坐月子期间切记不宜受凉。
  3. 产妇胃肠功能较虚弱，尽量吃一些容易消化的食物。
  4. 用热水浸湿毛巾，每隔两小时清洁恶露。
  5. 为了让膀胱恢复正常，分娩6小时内要排出小便。
  6. 产后2—3小时给孩子喂奶。
  7. 为了利于子宫收缩，回病房后24小时内要开始走动。
  8. 没有食欲也一定要吃饭！刚刚生产后身体疲惫，需要充分的营养和睡眠。

到了医院应该怎么做呢？

  1. 如果孕妈是先见红和宫缩的话，到了医院就去找帮你产检的那个科室医生，告诉她你目前的情况，然后医生会开住院单和一系列的检查单子，等把费用交齐了，孕妈就把待产包放到病房里，然后就去做检查，最后等待宫颈口开全。

到了医院需要办理什么手续呢？

  1. 挂号： 挂急诊号 ，急诊医生会给你做胎监和指检，如果胎监显示出现规律性宫缩或者开指，就会同意收你入院。

  2. 交费住院：住院前，需要向医院支付一部分费用，包括床位费和待产的费用（我当时一次性先交了8000，每个医院有差异）。交费之后拿着入院通知书和单据进病房。每隔一段时间会有护士来帮您指检，一般开到二指之后可以进待产室。如果医院人少，也可能直接进待产。如果需要打无痛，需要跟护士沟通联系麻醉师并且签协议书。

  3. 进产房：进产房前会签署一大堆协议，护士会指导。首先会进待产室，等你开指开得差不多了，再进产房生产。

  4. 生产结束之后，留观2小时，会有护士推你回病房。

坐月子

42天坐月子日程表，很全很重要！

无论是顺产还是剖腹产，都会让妈妈元气大伤。所以月子期里的调养真的很重要，要保证充足的睡眠，还要吃得好。

要注意的是，在初乳还没有下来之前千万不要吃任何催奶的食物，在乳房没有疏通之前，过早的催奶反而会引起乳腺堵塞。

科学坐月子

  1. 加强身体锻炼，练瑜伽等
  2. 使用姜水/糯米酒水泡脚或擦身体或洗头「艾叶」
  3. 月子里可以开空调，只要不对着吹，26-28度最合适；可以刷牙、吃常温水果
  4. 多走动：有利于产后修复
  5. 生娃一周后可以洗头、洗澡，注意不要吹风着凉，选择淋浴，禁止盆浴，不及时清理恶露，可能会感染
  6. 月子里不要喝油脂过多的猪蹄汤，多喝会使乳腺管堵塞
  7. 产后推荐食物：小米粥、稀粥、汤面、馄饨、豆浆、牛奶浆等，小米中含有铁元素和B族维生素，都是新妈妈产后所需要的，有利于体力的恢复。
  8. 有意识的做产后恢复最有效的就是每天坚持做凯格尔运动°！对盆底肌有很好的恢复作用，而且方便实用，坐着、躺着都可以练习，一般来说产后十来天就可以做了。

剖腹产注意

剖腹产手术后6小时可进食米汤、藕粉等流质软食，手术后一周内都禁食牛奶、豆浆、鸡蛋、蔗糖等胀气食品。等妈妈肠道排气后，可进半流质食物，如粥、肉汤等。

饮食：荤素搭配很重要，蔬菜水果不可少，稀粥蛋羹汤类可以弥补生产时流失的血和汗。

月子进补慢慢来，不同阶段饮食要点也要做一些调整。

  1. 月子第一周饮食：本周是代谢排毒阶段，食谱多以流食、半流食、汤水为主，多吃蔬果谷类，尽量保持食材原味，多喝汤水。剖产妈妈要等第二天排气后才能吃东西，在此之前用静脉输液。

    粥：小米粥、菠菜鱼片粥、猪肝粥、八宝粥等；

    汤：鲈鱼汤、猪肝汤、花生猪蹄汤、黄芪四神鸡汤、山药莲子排骨汤等

  2. 月子第二、三周饮食：本周是收缩内脏阶段，可用一些催奶的食谱。可以开始吃保养餐和补药。

  3. 月子第四周饮食：本周是滋养进补阶段，饮食上要注意多样化，多吃水果、蔬菜、谷物，并吃适量的肉、蛋和奶类，注意食品卫生。

  4. 月子第五、六周饮食：本周是调整期，保持健康的饮食即可。母乳妈妈在饮食上要注意多样化，多吃水果、蔬菜、谷物，并吃适量的肉、蛋和奶类，注意食品卫生。

饮食要点

  1. 喝足量的水

    一定要保证摄入足量的液体，每天要喝6—10杯（每杯250毫升）液体。

  2. 两餐之间吃点零食，别饿着自己

    母乳妈妈会经常容易饿，一天吃个5/6餐好正常，所以一定要准备一些小零食，饿的时候吃点东西，这也可以对抗疲劳。

    简便而营养高的零食包括：

    • 新鲜水果如：一只香蕉或一个苹果；
    • 无加糖、非油炸的干果(约30克)：葡萄干、杏脯、西梅；
    • 原味焗果仁(一中式汤匙满) ：核桃、杏仁、花生
    • 西红柿低脂芝士三文治(一片全麦面包+芝士1片+西红柿2片)；
    • 低糖的早餐谷物(一小盒或半碗)；
    • 鸡蛋一只；
    • 低糖加钙豆奶一盒；
    • 低脂奶酪一盒；
    • 低脂或脱脂奶一盒。

  3. 补充足够的矿物质

    产后妈妈要通过食品来摄入足够的矿物质，比如铁和钙。铁主要存在于瘦红肉、鱼、全麦粉面包、菠菜和豆子等。含钙丰富的食物有：奶制品、豆腐、绿叶蔬菜。

饮食禁忌

  1. 避免高脂肪、高糖分，免得增加额外的体重。

    喝汤的时候，可以用吸管，喝油层下面的汤水。吃肉的时候，去掉皮和肥肉部分。每天最多使用5~6茶匙油。少吃饼干、蛋糕、油炸食品。

  2. 母乳妈妈不要吃任何含有酒精的食品。

    任何含酒精的饮料、食品，都会通过血循环进入乳汁，改变乳汁的味道，这将使婴儿感到不舒服。如果酒量过多，还会影响宝宝发育，所以，母乳妈妈要滴酒不沾。

  3. 母乳妈妈要少喝咖啡、茶水、可乐。

    如果产妇喝一杯咖啡、或茶水、或可乐，乳汁中的咖啡因含量将很快达到高峰。婴儿不能像成人那样排出自己体内的咖啡因，容易造成体内咖啡因的积聚。

    饮用咖啡较多的母乳妈妈，会让宝宝睡眠不好，容易发怒。有喝咖啡、茶水习惯的母乳妈妈，每天饮用的量不能超过一杯，并在哺乳前2小时饮用。

怎样判断宝宝是不是厌奶呢

厌奶的表现有：

  1. 醒着不肯喝奶，要睡觉了迷糊了才吃。
  2. 白天吃的很少，晚上频繁醒来要吃。
  3. 看到乳头就哭，横着抱就打挺，硬塞奶头就哭。
  4. 吃吃停停，很容易被影响，听到一点声音就不喝奶了

厌奶很常见，厌奶的持续时间也有长有短，大部分宝宝持续2-3周时间就会自行调整过来，也有的宝宝会持续1个月。

千万别把厌奶当成是宝宝断奶的“信号”！1岁以下的宝宝，很少有主动断奶的。

婴儿

  • 6个月以下的宝宝，一旦翻身趴在床上，自己无力起身，他们的口鼻会被被子堵住，非常容易导致窒息死亡。
  • 产后最初几天吃得少，也没怎么活动，就便秘了！
  • 婴儿指甲生长较快，建议一周修剪一次，可在婴儿睡觉时修剪
  • 奶粉：宝宝在进行段位转换时，原则上需要进行转奶，转奶期间，可能会出现一些腹泻、便秘、不爱吃奶、哭闹等不适症状。
