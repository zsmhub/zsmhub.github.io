围棋的英文名有三种：中国读 Weiqi，西方读 Go，韩国读 baduk(发音：怕度)。

围棋对局一般分为三个阶段：布局、中盘、收官。收官阶段也称为官子阶段。然而，这三个阶段之间并没有明显的界限，何时进入中盘，何时进入收官，完全取决于棋局的发展和对局者的决断。

围棋中盘战术的主要技术，除了包括计算能力、掌控全局能力和形势判断之外，还包括进攻、防守、手筋、死活、打入、侵消等多方面的战术。

李昌镐(hao)说过一句名言，“围棋不是看你学了多少，而是看你会用多少。”

初学者

初学者注意：千万不要下简单人机，否则会被胖揍到失去自信，一定要和真人下！因为人会失误，而AI不会失误！

初学者可以先去下9路棋盘，9路棋盘主要是练习对杀、死活和官子等基本功。

9路棋盘虽小但变化复杂，开局优先占44、45，其次是55。【黑先手一般先占领天元】

围棋礼仪：执黑一方将第一手棋下在自己的右上角，因为这是伸手最方便的一个地方，同样也将对手最方便的地方留出，“这不仅是礼貌，还是换位思考”。

终局计算：数子法和比目法

当今世界，主要玩围棋也就中日韩三国，其中中国采用数子法，日韩采用比目法。

日韩比目法只要黑棋贴6目半，而中国数子法要黑棋贴3又3/4子。

总结一下，不管是按中国数子规则，还是日韩点目规则，对围棋胜负的判断结果是一样的，大家可以放心地交换使用。但是要说这俩完全一样的话需要有一个很苛刻的条件，就是黑白双方的手数一样，也就是说不能是黑棋收后，也不能有任何一方停招。

规则简单说明

数子法：

如果对局双方是平下，则终局后先将双方死子全部清理出盘外，然后对一方的活棋（包括活棋围住的点）以子为单位进行计数。 双方活棋之间的空点各得一半，一个点即为一子。 就现行的中国围棋规则来看，黑方超过184又1/4为胜，反之为负(注：平下不可能是和棋）。 (184.25 = 180.5 + 3.75)

如果是让子棋，则黑方在终局时还给白方让子数的一半，随后数棋。若本局是黑收后，则黑棋要超过181子算胜。若本局是白收后，则黑棋要超过180又1/2子算胜（注：可以是和棋）。

  1. 单官收完。双方确认无误，同意棋局终了。
  2. 然后剔除盘面上不活的孤子。
  3. 再来在双方的领地上，填满己方的棋子。
  4. 数双方的棋子数目，再加上规定的贴子数目，就可以判定输赢。

比目法：

不用把死棋去除，而是将棋放入本方空内（如：白方空内的死黑棋放入黑方空内）。将每方空内的空格数（即无子处）相加，判定胜负（目前规则是黑贴6目半）。

  1. 官子收完，领地划分完，双方确认无误，同意棋局终了。（可不收完单官）
  2. 然后将对方不活的孤子，填入对方的领地。
  3. 再把对方死子也同样的填入对手的领地。
  4. 最后数双方所围的空，再加上规定的贴目数目，就可以判定输赢。

规则详细说明

终局算法 黑 白
数子法 黑活子+黑空 白活子+白空
比目法 黑空-黑死子 白空-白死子

对两列表格分别做减法就会发现，比目法其实是在数子法的基础上减去了（活子+死子）。而活子+死子其实就是一共下在棋盘上的手数。也就是说，数子法得出的结果和比目法一样当且仅当黑白双方总手数一样。由于围棋是一人一手下的，因此双方的总手数应该是差不多的，所以我说这两种数法大体上是一样的，所以您一般不用担心因为换了一种数法结果就不同了。由于比目法得出的数字更小，所以对局时的形势判断一般都是用比目法。

传统打谱方式介绍

什么叫打谱？

打谱，就是按照高手下出的棋局的棋谱，把棋子在棋盘上按顺序再摆一遍。打谱过程中，可以感受高手下出棋的奇妙，可以停下来，自己研究，可以培养自己的大局观，等等，打谱也是长棋的一种方式。

有效的打谱方法

一、快速打谱

就是你照着棋谱能摆多快就摆多快，这种方法可以提高你的棋感。一段时间之后你会觉得下棋时很多地方一眼就找到了要走在哪里才好。但要注意这种方法一定要选择顶尖高手的棋谱，并且这个棋手的风格应该是稳健的，

二、超慢速打谱

这种打谱方法就是要详细的研究每个变化，每一步棋。这种方法选择的棋谱就可以是任何棋风的了。只要是喜欢想研究的棋手的都行。

打谱的目的

  1. 学习对局者一些着法，主要是战术方面；
  2. 学习一些布局知识、定式和套路，特别是当下较流行的具体走法；
  3. 入门者通过打谱，模仿高手的序盘行棋步调，从纠缠局部开始转到把棋走开。
  4. 通过一手执书，一手落子，坐直身姿，气定神闲来提升个人的气质和休养。

打谱的方法

  1. 根据打谱人的水平，由少至多的来打。初学者可以摆50手左右；中级水平可以打100手左右，高级者可以通盘来打。
  2. 谱中的解说一般不用全部细看，最好是遇到不甚理解的着法时，可以看解说来解惑。

围棋术语

手筋

手筋，围棋术语，出自日本，意思是局部对弈中的最佳妙招。日本棋道中的“筋”与中国的原意最为接近，为“精髓的”、“要害的”之意。棋中有“宁舍片子，不舍一筋”的说法。

在围棋中，“筋”意指棋子与棋子之间的关系。在所有的筋之中，效果极大的特殊手法，即称为手筋。

常见手筋：双打、枷、扑/倒扑、接不归[扭羊头]、挂、尖、长（立、冲、退、爬、贴、并、弯）、扳、粘、跳、飞、挡、碰（靠、压、托、盖、顶）、虎、挖、挤、拐、跨、镇（封）、穿象眼、刺（点、觑）、夹、接、拆、逼、渡、搭、吊

官子

「单官」，就是占不到“目”的一手棋，只是用棋子占据棋盘上的一个交叉点。单官一般都是最后才去抢占的。

根据黑方或白方谁先落子，以及落子后谁是先手谁是后手，官子可以分为4类：「双先官子」「单先官子」「逆先官子」「双后官子」。

收官时机的把握

  1. 有紧急问题时不宜收官：当棋子有死活、连接、安全隐患的前提下，要先处理
  2. 有重要大场时不宜收官，如先抢拆二

正确的收官顺序

  1. 双先单先无限抢，逆先双后看价值
  2. 先三后二再看一，三线看拆二、拆一，二线看小尖、小飞，一线看扳粘、大飞。

追求价值的行棋理念

  1. 走在目上，尽可能让自己的棋子一子多用
  2. 高效补棋：补外不补内

常见官子价值

  1. 普通消劫价值 1/3 目
  2. 一路仙鹤大伸腿价值约 9 目
  3. 一路扳粘价值约 4 目
  4. 二路扳粘价值约 12 目

战术

弃子战术：弃子整形、弃子争先、弃子求活、弃子转换、弃子取势、弃子包扎、弃子收气等等

声东击西战术：不直接攻击弱棋，通过攻击强棋的方式顺势攻击弱棋

寻找缺陷：寻找断点（断打，双断），寻找气紧，寻找死活

其他术语

妙手、本手、俗手、恶手、缓手、鬼手

围空、外势/厚势，实地、抢先/脱先、禁着(zhuo)点、真假断点、愚型与裂形

模样：所谓模样，指尚未形成确定地域的大阵势。

净活：若干子虽被包围，但能做成两个（或两个以上）眼，无条件地成活，称为“净活”。净杀：即净吃，无法成活。

效率：围棋最终比较的是双方领地的大小，所以在对局中，每一手棋都是以多占领一些领地为目标。一手棋在围空上发挥的作用越大，我们就说这手棋的效率越高。如果能在某个重要的局部连下两手棋，那么这两颗棋子的效率通常会成倍增加。

急所：来源于日本围棋术语，现在我国棋界也广为使用了。它是指在敌我双方互相接触的棋形中，无论对于攻方还是守方都互关重要的那些点。

缓招：在围棋中，步调慢，效率不高的棋叫做缓招，缓招一般可以暂时不走，也就是不是“急所”。实战中走出缓招，会被对手抓住机会，痛下杀手的。

计算力和大局观，是下好围棋的两大技能。计算力，是指遇到局部战斗时通过对下法的推算找到局部最佳下法的能力。吃子、死活和对杀的知识，都是计算力的核心部分。大局观，是指在全局范围内通过对各个部分的价值判断找到全局最佳下法的能力。布局、定式、中盘和官子的知识，都是大局观的核心组成部分。

孤棋，围棋术语，对局中，一方在对方势力的威胁下尚未生根安定的棋子。治孤，就是巧妙利用对方的棋型缺陷和薄弱环节，将自己的孤棋进行妥善、高效的处理，“治孤法”是一种围棋高级战术。

勺子（漏勺）：指的是围棋对局中出现明显漏招，误算，招至不必要的损失。

常见扭十字的处理方法：1. 中央扭十字，玩转大风车 2. 边线扭十字，记得长三线 3. 角部扭十字，下立最简明【周边有配合时的攻击手段：打了贴】

滚打包收：利用断、扑、挖、枷等手段，将敌子围成凝重棋形的着法，就叫做滚打包收

金鸡独立：对手左右两边不入气（小技巧：找断点，左右两块棋都是只有两口气的情况）

宽枷：复杂的枷吃，在正常枷吃的位置向自身薄弱棋子移动一个格子(小跳变成小飞)，保护自己气短的棋子

宽征：漏着气的征吃，技巧：一路推到底，二路拐个弯

回龙征：利用周围子力关系滚打包收，使对方气紧然后再进行征吃的技术。

聚杀：指攻杀对方的棋时，采取点入对方空间的手法，通过巧妙的次序，使下的一系列棋子在对方腹地形成弯三、直三、方四、丁四、刀五等棋形，对方即使把这些子提掉，棋形还是存在破绽，己方可以通过再次点入、扑等手段，杀掉对方的棋子

胀死牛：通过打吃的手段给对方棋子制造不入气从而做活的做活方式，常用手段：扑

倒脱靴：角部做活手段，通过制造方四或闪电四的棋型送给对手棋子吃，然后再接不归反吃对手的棋子

拆边的应用技巧：1. 窄边：头低少拆，头高多拆。2. 宽边：拆边不过五，拆边留拆二。

天王山：消涨的要点

象步卡眼、双板凳、靠单

棋子的强弱、厚薄、轻重

相思断：断的一种，名为断，其实是为了连接。无论对方怎样应接，其重要棋子最终仍无法摆脱被歼的困境。

棋筋与废子：棋筋：能分断对方的棋子；废子：作用不大，即使弃掉也不会有明显损失的棋子。

禁全同：禁止全局同形再现，如劫

劫的种类：先手劫/后手劫，无忧劫，紧气劫，天下劫，两手劫，三手劫，缓一气劫，缓二气劫，缓三气劫，万年劫，连环劫（双活），三劫循环（无胜负），长生劫(会恢复原样)

利用不入气：吃子、破眼、做眼

围棋棋理

可分 攻、防、围、破 四个方向进行讲述围棋棋理。

  • 棋理提供行棋方向，计算得出具体选点（对局技巧：多思考分断对手和利用不入气）
  • 下棋思考顺序：先处理不安定的棋，再下宽广的棋
  • 学会观察棋形，棋形的要点：在断点比较密集的地方或在对方要做眼的位置
  • 二子头必扳或必长，三子头必扳或必长
  • 要有气紧的意识，四五六：四口气就是气紧，五口气就是快气紧，六口气及以上就没事
  • 杀棋：逢方必点（做眼的要点，需要一些想象力）
  • 急所：发展急所，攻防急所，消涨急所，双方互为拆一为急所
  • 三思而行：道生一，一生二，二生三，三生万物。此三为多之意。围棋的本质是三步计算法，每手棋至少计算3步（即每手棋必须考虑3个及以上的点进行计算），然后对比，选择获利最多的变下
  • 厚势不围小空：厚势围空厚上加厚，薄棋围空弱棋变强
  • 不得贪胜：优势局一般优先考虑防守，处理自身缺陷，再考虑进攻扩大地盘
  • 弱的棋子可以贴着对方厚的棋子行棋，这样更容易走强，如飞靠，飞尖
  • 活棋之前对对方的弱棋进行冲击，然后再活棋
  • 走双活的时候，要让自己的内部空间大一些，使对手无法聚杀（有眼双活/无眼双活）
  • 两打同情不打，推敲扳虎兼长（同情：同种情形）

进攻与防守

「分断」、「封锁」、「搜根」、「打入」和「侵消」是进攻的主要手段。

  1. 「分断」是指的是把对方的棋形分割成两块或更多的棋，使对方失去连络和气息，常见的分断手筋有扳断、扭断、尖断、挖断、搭断、冲断、跨断

  2. 针对孤棋，「封锁」是指堵住孤棋向中腹逃跑的方向，将孤棋困在角部或边上

  3. 针对孤棋，「搜根」是指缩小或破坏孤棋在角部或边上的做眼空间，使孤棋无法做成两只真眼

  4. 无孤棋情况下，「打入」是指深入对方的地盘，成为对方地盘中的孤棋，但可以彻底破坏对方的地盘

  5. 无孤棋情况下，「侵消」是指压缩对方地盘的范围，阻止对方扩张，使对方只能采取巩固的方法围空

「补断」、「出头」、「生根」、「巩固」和「扩张」是防守的主要手段。

  1. 「补断」是指在自己的断点上落子，使自己的棋连成一片，常见的补断手筋有接、虎和飞

  2. 针对封锁，「出头」是指将孤棋向中腹延伸，保证逃跑路线的畅通

  3. 针对搜根，「生根」是指扩大在角部或边上的做眼空间，使孤棋做成或随时可以做成两只真眼

  4. 「巩固」是指堵住自己地盘的缺口，防止对方入侵

  5. 「扩张」是指将自己地盘的范围扩大，争取围住更大范围的地盘

手筋题和死活题

做死活题的规律，一般是一冲、二扳、三点杀。

手筋题和死活题的主要区别有三个：

一是开放性不同，死活题往往限定在一个封闭的局部，可选的点一般不多；而手筋题往往给出的是开放的区域，选点通常要比同等难度的死活题多，因此在缺乏经验积累的情况下，需要考虑的点位就多。

二是思维步骤不同，死活题预先给出了目标，只需要考虑做活或者杀棋的手段就行了，不需要有任何战略性的考虑；而手筋题先要评估得失，然后根据对局面的评估选择目标，比如选择逃出，还是弃子之类，然后才是考虑具体的手段。即使有些题目为了降低难度，在题目分类上指明了手筋的类型，比如说封锁的手筋，但具体应该采用什么手段去封锁，很多时候依然要根据局面去评估。潜在选点可能更多，思考步骤也更多，有时候觉得无从下手也就很正常了。

三是目标不同。这一点其实是最重要的。做死活题的时候，虽然顺便也会记住一些常见死活，但最主要的目标是培养计算力；而手筋题的最终目标其实是培养第一感——虽然一开始会觉得选点众多，但随着经验的积累，看到类似的局面时，应该可以立即把选点缩减至很有限的几个，然后根据对全局的判断，仅在这几个选点中进行计算。而这虽然并不提升你的计算力，但会大大提升你的计算效率，从而进一步提升你的计算深度——在后续计算的每一步中，都减少在无效选点上的计算消耗。

死活手段

  1. 杀棋手段：缩小眼位，破眼（点杀，打劫杀）。常见手筋：聚杀，滚打包收；破眼手筋：倒扑、制造倒扑。
  2. 做活手段：扩大眼位，保护眼位，寻求接应。常见手筋：胀死牛、接不归(技巧：打三还一)、倒脱靴(用聚活这个词挺形象的)。

12种基础死活棋形

  1. 2种死形：直二，方四
  2. 4种活形：直四，曲四，闪电四，板六
  3. 6种半死不活形：直三，弯三，丁四，刀把五，梅花五，葡萄六

实战死活常型

  1. 边线死活常型

    • 直线型：六死八活七看先，直接缩小眼位
    • 虎口型：已封口直接破眼，未封口优先缩小眼位。

  2. 角部死活常型：

    • 直线型：四死六活五看先，直接缩小眼位
    • 虎口型：角部直二直三优先缩小眼位，直四直接破眼，。
