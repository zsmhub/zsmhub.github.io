快捷键

  1. command+option+i: 打开 iterm2
  2. command+option+c: 打开chrome控制台 【可自行修改为F12】
  3. command+N: 打开第二个qq，command+T: 打开第二个其他应用
  4. cmd+shift+backspace: chrome清除缓存快捷键
  5. cmd+`: 同一应用多窗口切换
  6. cmd+tab: 不同应用切换
  7. cmd+空格: 聚焦搜索
  8. 文字拖拽：光标移到开头后，按下shift键再移动光标，即选中一段文字

常用操作

  1. 删除启动台的一些顽固图标

    方法很简单，打开访达（Finder），前往 - 前往文件夹（或快捷键 Shift+Command+G），输入路径：/Users/Insomnia/Applications，选中里面的那个顽固App，点击右键，将图标移到废纸篓即可。

  2. MacOs 自带 apache

    • 启动命令：sudo apachectl start/restart,
    • 停止命令：sudo apachectl stop，
    • 配置文件路径：/etc/apache2/httpd.conf

  3. 显示隐藏文件：在终端输入以下命令：

    defaults write com.apple.finder AppleShowAllFiles -boolean true; killall Finder

  4. finder文件夹显示文件路径

    defaults write com.apple.finder _FXShowPosixPathInTitle -bool TRUE; killall Finder

  5. 修改环境变量方式

    • vim ~/.profile 【生效需重启电脑】
    • vim ~/.bash_profile【生效只需执行source ~/.bash_profile】

  6. 配置 subl 启动 sublime text 编辑器：做软链

    sudo ln -s /Applications/Sublime\ Text.app/Contents/SharedSupport/bin/subl /usr/local/bin/subl

  7. MacOs 读写U盘方式

    在“LaunchPad－其他－硬盘工具”里，选择移动硬盘－擦除－设置文件格式为 ExFat，这个文件格式是Windows和Mac都可以读写的。这个文件格式也没有文件和硬盘大小的限制，硬盘有多大，就能存储多少东西。

好用的应用

  1. Yoink

    是 Mac 上一款临时文件暂存的工具类 App，它就像是一个「中转站」，让你能更轻松地把文件从一个窗口移动到另一个窗口。

  2. Parallels Desktop

    虚拟机软件，可以在macos上很流畅的运行windows系统

  3. The Unarchiver

    解压缩软件

  4. SourceTree

    一款免费的Git客户端

  5. Proxyman(免费)、Charles(收费，手机使用UC浏览器访问 chls.pro/ssl 下载CA证书，其他浏览器可能无法下载证书)

    抓包软件

    工具 | Charles对Chrome浏览器抓包

  6. RescueTime

    时间管理工具，记录你在chrome浏览器以及电脑各个app的使用时间。

  7. MindNode

    绘制思维导图工具

  8. CheatSheet

    专门用来查看应用程序快捷键的工具，安装好cheatsheet软件后，在软件界面长按Command键即可快速查看快捷键列表，非常的便捷。

  9. Send Anywhere

    Send Anywhere 是一个免费的跨平台传输软件，使用不受平台和文件大小的限制。

  10. Wattagio

    是一款功能强大的电量管理软件，不但拥有常见的电量监控，电池性能检测，还可以读取原始电池数据，提供给用户参考，并且将其制作成“history”图表更加直观的看到电池性能表现，但更重要的是它可以监控每个软件的电量消耗状态，并在 Dock 软件图标的底部显示不同颜色的线条，以示对应的用量级别。

  11. Krita

    是一款免费和开源的的图像处理软件，是photoshop的替代者。

    官网下载免费

  12. snipaste

    截图+贴图神器 https://zh.snipaste.com/

  13. Whisper

    如果你有跟我一样的视频识别生成字幕的需求，或者做播客、视频会议需要转录成文字，也可以试试这款软件。

    本地音乐播放器，在 apple store 下载，支持苹果账号登录。

2
3
4

killall Dock
sudo mount -uw /

压缩包乱码

将Mac上的压缩包传输到Windows，打开后文件会出现乱码。

原因：Mac下默认文字编码是utf-8，而在WIN下是gbk。

解决方法：将 .zip 的压缩包后缀名改为 .rar 或 .7z 格式，然后解压。

如何删除 MacOS 启动台多余的/残留的图标

1
2

# 终端执行以下命名即可，注意修改对应图标的名字
sqlite3 $(sudo find /private/var/folders -name com.apple.dock.launchpad)/db/db "DELETE FROM apps WHERE title='你要删除图标的名字';" && killall Dock

文件按空格键无法预览

重启 Finder 一般能解决，提供两种重启方法：

  1. 按住 “Option” 键和 “Command” 键再按下 “Esc” 键，选中 “Finder”，然后点击 “Relaunch”。
  2. 按住 “Option” 键，在程序坞的访达图标上点击鼠标右键，选择 “重新开启”。
