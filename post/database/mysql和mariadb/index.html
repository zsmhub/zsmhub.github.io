Mysql 和 MariaDB 的关系

MariaDB 数据库管理系统是 MySQL 的一个分支，主要由开源社区在维护，采用GPL授权许可。

开发这个分支的原因之一是：甲骨文公司收购了 MySQL 后，有将 MySQL 闭源的潜在风险，因此社区采用分支的方式来避开这个风险。

MariaDB 之于 MySQL，类似于 CentOS 之于 Redhat。由于 Oracle 拥有 MySQL 的名字和海豚商标，基于此如果各公司继续用 Mysql 可能存在法律纠纷，于是纷纷开始选择兼容性好而且研发力量强大的 MariaDB。
