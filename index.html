肚子瘦下来

2026-01-20
生活笔记
约 738 字 预计阅读 2 分钟

男性“肚子变大”，本质上通常是内脏脂肪增加 + 肌肉量下降 + 激素与代谢变化的综合结果。

必胖肚子杀手

  1. 蛋白质不足 = 肚子最容易囤脂肪
  2. 睡眠 < 6 小时 = 必胖肚子（睡眠不足 → 睾酮下降 → 脂肪更爱囤在腹部）
  3. 长期压力 = 内脏脂肪加速器（皮质醇升高，脂肪优先堆在肚子）
  4. 酒精（尤其啤酒）是中年肚腩第一杀手（一周不超过 2 次，每次不超过 1 杯）
  5. 久坐是腹型肥胖的“加速器”，建议每隔 1 小时起身活动 5 分钟，拉伸腹部和腰部肌肉。

饮食：这是决定性因素（占 60–70%）

费用统计时间：2026年1月10日

国外费用

  1. sora2：支持10~25秒，标准版一个视频0.7元，高级版3.5元/个
  2. veo3： 支持4~8秒，标准版1元/秒，高级版2.8元/秒

国内费用

  1. 通义万相：支持5~15秒，标准版0.6元/秒，高级版1元/秒
  2. 即梦：支持5~12秒，1元/秒，2并发；API不支持最新的3.5Pro模型
  3. 海螺：支持6~10秒，标准版一个视频>=2元，高级版>=3.5元/个
  4. 可灵：支持5~10秒，付费模式只能购买资源包，3万起步，有效期3个月

发现的问题

  1. AI视频对中文支持不友好，商品上的文字容易乱码
  2. 国内视频平台容易生成不符合逻辑的操作

AI视频在创作流程方案一

  1. 根据”xxx“主题生成x个分镜脚本
  2. 生成分镜图片：每一个分镜脚本生成分镜图的提示词，包含中英文
  3. 根据对应的分镜图，开始制作分镜视频
  4. 等待所有视频生成
  5. 所有视频合成剪辑

AI视频在创作流程方案二

  1. 帮我设计一个一分钟的搞笑故事脚本
  2. 根据脚本，给我分镜头的提示词
  3. 根据分镜头，给我详细的图片提示词
  4. 文生图
  5. 图生视频
  6. 剪辑合并成一个完整的视频

Node.js 是一个基于 Chrome V8 引擎 的 JavaScript 运行时环境，它允许开发者在 服务器端 运行 JavaScript 代码，而不仅仅局限于浏览器。

核心定位

  1. Node.js 不是新的编程语言，而是让 JavaScript 脱离浏览器运行的 运行环境（包含执行引擎、内置 API 等）。
  2. 核心价值：统一前后端开发语言（均使用 JavaScript），降低开发成本；同时利用 V8 引擎的高性能，实现高效的服务器端编程。
